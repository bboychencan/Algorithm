最后实现上有点小问题，在计算2^(j-i)的时候，超时了，然后发现pow(2, (j-i), mod)是一个高效很多的方法，学到了。

```python
pow(2, n, mod)

# 上面比下面这个快很多
2 ** n % mod
```
